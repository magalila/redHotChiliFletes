# Carpeta: public/js

## Propósito
Contiene los scripts del frontend para cada vista del sistema.  
Cada archivo maneja la lógica de interacción con el backend y la actualización dinámica del DOM.

## Archivos principales
- **login.js** → Maneja el inicio de sesión y redirección según rol.
- **register.js** → Controla el registro de usuarios y subida de archivos.
- **proveedor-dashboard.controller.js** → Controla la navegación del panel del proveedor.
- **proveedor-perfil.controller.js** → Obtiene y muestra datos del proveedor.
- **proveedor-moviles.controller.js** → Gestiona el formulario y listado de vehículos.

## Dependencias
- Bootstrap 5 para estilos y modales.
- Fetch API para comunicación con el backend.
