# Carpeta: frontend-proveedor

## Propósito
Contiene las vistas HTML del panel del proveedor (frontend).  
Incluye las secciones de perfil, móviles y dashboard principal.

## Archivos
- **proveedor-dashboard.view.html** → Vista principal con navegación lateral.
- **proveedor-perfil.view.html** → Muestra datos personales y documentos del proveedor.
- **proveedor-moviles.view.html** → Permite agregar y visualizar vehículos y sus documentos.

## Relación con otras carpetas
- Usa scripts de `/public/js/` para manejar la lógica del frontend.
- Se conecta con rutas definidas en `/routes/proveedorRoutes.ts`.
