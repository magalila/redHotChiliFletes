# Carpeta: utils

## Propósito
Funciones auxiliares y utilitarias que se usan en diferentes partes del proyecto.

## Archivos
- **fileUtils.ts** → Manejo de archivos (guardar, renombrar, validar).
- **dateUtils.ts** → Funciones para formatear fechas y horas.
- **logger.ts** → Registro de logs y errores.

## Relación con otras capas
- Se importan en controladores y middlewares para tareas comunes.
